# Mr. Fish 🎣

Discord-бот для рыбалки! Лови рыб, смотри топы, удивляйся описаниям и просто отдыхай)

## Команды

- `/fish` — поймать рыбу
- `/leaderboard` — топ рыбаков
- `/looserboard` — топ неудачников
- `/help` — подсказка

## Как запустить

1. Клонируй репозиторий
2. Открой терминал и напиши: dotnet run
3. Вставь токен бота в конфиг
4. Зови друзей и рыбачьте вместе!
